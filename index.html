ABAP Development - ⭐️⭐️⭐️⭐️⭐️

ABAP is a programming language developed by SAP for the development of business applications in the SAP environment. The ABAP Objects component makes object-oriented programming possible. A BADI(Business Add-In) is the object oriented method of enhancing an SAP system. It is an enhancement technique that facilitates an SAP developer to add some additional code to the existing program in the SAP system.

SAP Forms - ⭐️⭐️⭐️⭐️⭐️

Form printing covers creating and maintaining a form for mass printing in SAP systems. This includes design and layout of the form as well as form logic. Within SAP ERP as well as S/4 there are different types of forms used. SAP Smart forms or ADOBE Forms. In SAP S/4 HANA, the target architecture is based on Adobe Document Server and Adobe Forms only. It is recommended to use Adobe Forms but due to compatibility the form technologies known like Smart forms and Sap Script are supported as well.

ABAP on HANA - ⭐️⭐️⭐️⭐️★

SAP HANA is an in-memory, column-oriented, relational database management system developed and marketed by SAP SE. Its primary function as a database server is to store and retrieve data as requested by the applications. ABAP (development) for SAP HANA refers to all developments which leverage the power of SAP HANA within ABAP-based applications. In this context, the SAP HANA platform is the primary database underlying the ABAP platform. This system deployment option is supported with AS ABAP 7.4 onwards.

SAPUI5 - ⭐️⭐️⭐️★★

SAPUI5 uses HTML5, Javascript and CSS for developing user interface technologies in SAP HANA, NetWeaver ABAP, and Java systems. SAPUI5 provides a set of JavaScript and CSS libraries, which can be combined in an application using the combinations that are supported.
